# aGo First Run

First-run wizard for fresh WordPress installs: cleans up demo content and applies sensible defaults. Use once, then deactivate. **Free, no Pro tier.**

[English](#english) · [Español](#espanol)

---

## English

aGo First Run cleans up the demo content WordPress creates ("Hello World" post, "Sample Page", default plugins) and applies the configuration almost every site needs: clean permalinks, timezone, comments off, essential pages.

### Tasks

- Delete demo content (post, page, default plugins).
- Configure permalinks and timezone.
- Disable comments globally.
- Create essential pages (Contact, About) as drafts.
- Pick a static front page (Home) instead of latest posts.
- Delete unused themes (keep only the active one).
- Clean expired transients and orphan database rows.

### Installation

1. Upload `ago-setup` to `/wp-content/plugins/`.
2. Activate the plugin.
3. Go to **aGo Tools, First Run**. Select tasks and click Run.
4. When done, deactivate the plugin.

## Español

aGo First Run limpia el contenido de demo de WordPress (post "Hello World", pagina "Sample Page", plugins por defecto) y aplica la configuracion que casi todo sitio necesita: permalinks limpios, zona horaria, comentarios desactivados, paginas esenciales.

### Tareas

- Eliminar contenido demo.
- Configurar permalinks y zona horaria.
- Desactivar comentarios globalmente.
- Crear paginas esenciales (Contacto, Nosotros) como borradores.
- Pagina estatica de inicio.
- Eliminar themes no usados.
- Limpiar transients expirados y filas huerfanas.

### Instalacion

1. Sube `ago-setup` a `/wp-content/plugins/`.
2. Activa el plugin.
3. Ve a **aGo Tools, First Run**. Selecciona tareas y haz clic en Ejecutar.
4. Al terminar, desactiva el plugin.
